Pill Box Project Details

Pill Box

#art #lasercut #wood

A laser-cut solid hardwood pill box with sliding lid.

Description

This pill box is an attempt at a useful, approachable community technology product. It's made from solid Black Walnut, which is native to where I live in New York State. It's a beautiful functional object that is a joy to use, and which I hope to reproduce for others in my community, perhaps with personalized engraving and/or custom dimensions. Materials cost is < $5, and laser + assembly time is < 10 minutes.
