Description

I built this static website generator to maintain my personal website, derekenos.com.

It's relatively simple, has accessibility, SEO best practices, and structured data (via Open Graph tags and Microdata markup) built in, supports cloud object storage for large static files, and has a live-reload feature for more joyful local development.
